Proyecto: librero

¿Cuáles son tus libros favoritos? ¡Haz un librero con ellos!

  • Este programa solo despliega un libro. Haz un arreglo de libros (por lo menos dos) y usa un bucle para desplegar una fila de libros.
  • Agrégale una propiedad de autor a cada libro y despliega el autor debajo del título.
  • Agrégale una nueva propiedad a cada objeto para almacenar un color al usar el comando color y usa eso para rellenar cada libro de manera diferente. (¿O quizá puedas almacenar una imagen?)
  • Agrégale una propiedad booleana a cada libro (como si se lo recomendarías a una amigo) y usa una declaración condicional dentro del bucle para dibujar algo diferente dependiendo de si la propiedad booleana es verdadera o falsa.
  • Ahora agrega más libros y usa un bucle para dibujar más repisas hacia abajo del lienzo. Piensa acerca de cómo puedes usar declaraciones condicionales o el operador % para hacer eso.
¿No quieres hacer un librero? También podrías hacer una repisa de videos o de video juegos, o recrear la pantalla de incio de tu teléfono inteligente.
